## How to run the app

1. Install all needed dependencies:
   `npm i`
2. Build the app:
   `npm run build`
3. Make npm link in other project:
   `npm link custom-components-lib`
   or publish npm on this project to use it by npm in others 
   `npm publish custom-components-lib`
